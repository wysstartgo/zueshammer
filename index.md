# ZuesHammer - Super AI Agent

![Version](https://img.shields.io/badge/Version-2.0.0-blue.svg)
![Python](https://img.shields.io/badge/Python-3.10+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

The intelligent AI agent with local brain, voice interaction, and three-tier memory.

## Features

- 🧠 Local Brain - Pattern matching, learns new skills
- 🎙️ Voice-First - Whisper STT + Edge TTS
- 🧬 Three-Tier Memory - Short, long, and working memory

## Quick Start

```bash
git clone https://github.com/pengrambo3-tech/zueshammer.git
cd zueshammer
pip install -r requirements.txt
python3 src/main.py --mode cli
```
