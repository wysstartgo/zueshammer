Get Started in Minutes

Simple integration into your projects.

example.py 
from src import ZuesHammer
from src.core.config import Config

config = Config.from_default_locations()
config.permission_level = "semi_open"
config.voice_enabled = True

agent = ZuesHammer(config)

async def main():
    await agent.start()
    
    # Voice input or text
    response = await agent.process("Hello, help me analyze this project")
    print(response)
    
    await agent.stop()

main()
Quick Start

Get Up and Running

1

Clone

Download the repository

git clone https://github.com/pengrambo3-tech/zueshammer.git
cd zueshammer
2

Configure

Set up your API key

# China LLM (Recommended)
echo "CHINAWHAPI_KEY=your_key" >> ~/.zueshammer/.env
echo "API_PROVIDER=chinawhapi" >> ~/.zueshammer/.env
echo "MODEL=deepseek-chat" >> ~/.zueshammer/.env
3

Install

Install dependencies

python3 install.py

# Or with pip
pip install -r requirements.txt
4

Run

Start the agent

# Voice mode (recommended!)
python3 -m src.main --mode voice

# Or CLI mode
python3 -m src.main --mode cli
