Core Features

Three Breakthrough Technologies

Unlike typical AI agents, ZuesHammer combines unique technologies that make it smarter, faster, and more efficient.

🧠

Local Brain

Intent recognition and pattern matching that learns new skills automatically. 80% of common tasks run instantly without calling LLM.

🎤

Voice-First

Local Whisper STT + Edge TTS for true hands-free operation. No data sent to cloud - fully offline capable.

🧬

Three-Tier Memory

Short-term (LRU cache) + Long-term (SQLite) + Working memory. Never forget important context.

🌏

China LLM Support

Unified API via chinawhapi.com for DeepSeek, Qwen, GLM, Moonshot, ERNIE, Doubao, MiniMax.

Quick Start

Get Up and Running

Follow these steps to set up ZuesHammer in minutes.

1

Clone

Download from GitHub

git clone https://github.com/pengrambo3-tech/zueshammer.git
cd zueshammer
2

Configure

Set up your API key

# ChinaWhapi (recommended)
echo "OPENAI_API_KEY=your_key" >> ~/.zueshammer/.env
echo "API_BASE=https://api.chinawhapi.com/v1" >> ~/.zueshammer/.env
echo "MODEL=deepseek-chat" >> ~/.zueshammer/.env
3

Install

Install dependencies

python3 install.py

# Or with pip
pip install -r requirements.txt
4

Run

Start the agent

# Voice mode (recommended)
python3 -m src.main --mode voice

# Or CLI mode
python3 -m src.main --mode cli

One-Line Install

Run this command to install ZuesHammer with all dependencies:

$ curl -sSL https://raw.githubusercontent.com/pengrambo3-tech/zueshammer/master/install.sh | bash
